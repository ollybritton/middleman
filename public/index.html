Middleman

Middleman is a tool that allows you to see websites from the perspective of a computer in France. The computer will fetch the website for you and then display that version to you.

Use

To use, create a bookmark with this link or using this URL:

javascript:window.location.replace("http://middleman.ollybritton.com/url/"+window.location)

When you wish to see the website from a different perspective, click the bookmark.
